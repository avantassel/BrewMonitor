API Documentation

GET /brewmonitor/api/v1.0/list

  • Returns contents of brew -> tempData in JSON
  • Takes URL parameters, e.g.
    • /brewmonitor/api/v1.0/list?devid=1&userid=1

POST /brewmonitor/api/v1.0

  • Insert into brew -> tempData
  • { "userid": 0, "devid": 0, "temps": [{"sensid": 0, "temp": 20.0}, {"sensid": 1, "temp": 20.0}], "timestamp": "2015-02-03T16:52:08.506Z" }

GET /brewmonitor/api/v1.0/help

  • Returns list of API methods and formats
